Bioinformatics with Rosalind - RNA

July 23, 2020

1 minute read

 LinkedIn

 Email

This Rosalind problem deals with RNA. RNA differs slightly from DNA, in that the ribose is oxy’d instead of deoxy’d, and instead of (T)hymine is has (U)racil. The problem itself is pretty simple:

Given: A DNA string t having length at most 1000 nt.

Return: The transcribed RNA string of t.

My answer is as follows. Note this time I load in the string from a text object instead of copying it into the program itself, and output the answer into a text file as well.

DNAsequence = open('rosalind_rna.txt', 'r')
RNAsequence = DNAsequence.read()
with open('output.txt', 'w') as output:
    output.write(RNAsequence.replace('T', 'U'))

And there it is, my four lines of pure bioinformatic power!
